# AI Agent Architecture Substack Newsletter

Weekly technical deep-dives into autonomous AI agent design from a team that builds them.

## Overview

This repository contains the public archive for the *AI Agent Architecture* Substack newsletter. Each week, we publish a technical deep-dive covering autonomous AI agent design patterns, infrastructure decisions, and real-world performance data. We publish from our own experience building and running the Kubera autonomous agent system — genuine first-party content that no human blogger can replicate at this cadence.

## Quick Start

This repository is an archive of newsletter outlines and public articles. To get started:

1.  **Browse the Archive:** Review the available markdown files in the repository to see past public content.
2.  **Subscribe for Free:** Visit our [Substack page](https://kuberaai.substack.com) to subscribe to the free weekly newsletter.
3.  **Explore the Code:** For the complete implementation code and architecture diagrams, upgrade to the Premium version (details below).
